## 📁 Projects

### 🔑 Password Generator
A secure and customizable password generator built with Python.
- Generates strong passwords with custom length and character sets
- Uses OOP principles for clean, reusable structure
- **Tech:** Python, OOP

---

### ✂️ Rock Paper Scissors Game
An interactive CLI-based game with score tracking.
- Player vs Computer with real-time score tracking
- Replay functionality and randomized computer moves
- **Tech:** Python, Random Module

---

### 🧮 Calculator Application
A fully functional calculator with input validation.
- Supports all basic arithmetic operations
- Handles edge cases and invalid inputs gracefully
- **Tech:** Python, Error Handling

---

### 🎯 Number Guessing Game
A fun guessing game with difficulty levels.
- Multiple difficulty modes (Easy / Medium / Hard)
- Limited attempts with hints system
- **Tech:** Python, Random Module, Loops

---

### 🎲 Dice Rolling Simulator
A multi-dice simulator with result tracking.
- Simulates rolling multiple dice simultaneously
- Tracks and displays roll statistics
- **Tech:** Python, Random Module, Functions
